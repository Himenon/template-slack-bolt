# @himenon/template-slack-bolt

## Development

**Install**

```bash
pnpm install
```

**Watch**

```bash
pnpm watch
```

**Build**

```bash
pnpm build
```

**Build Docker Image**

```bash
pnpm build:docker
```

## Release

- Automatic version updates are performed when merged into the `main` branch.
